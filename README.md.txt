# 🚀 Cosmic Clicker

An addictive space-themed incremental clicker game built with pure HTML, CSS, and JavaScript.

## 🎮 How to Play
- Click the planet to harvest energy
- Buy upgrades to automate energy collection
- Prestige to gain permanent multipliers
- Unlock achievements as you progress

## ✨ Features
- Beautiful space animations
- Multiple upgrade paths
- Prestige system for long-term progression
- Achievement system
- Responsive design

## 🚀 Play Online
[Play Cosmic Clicker](https://yourusername.github.io/cosmic-clicker-game)

## 📱 Local Development
Just open `index.html` in your browser - no setup required!

## 🛠️ Built With
- HTML5
- CSS3 (animations & gradients)
- Vanilla JavaScript
